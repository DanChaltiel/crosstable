# Test if an object is a crosstable

Test if an object is a crosstable

## Usage

``` r
is.crosstable(x)

is.transposed_crosstable(x)

is.compacted_crosstable(x)

is.multiby_crosstable(x)
```

## Arguments

- x:

  An object

## Value

TRUE if the object inherits from the `crosstable` class or other
subclasses.
