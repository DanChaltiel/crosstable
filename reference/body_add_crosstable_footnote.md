# Adds a standard footnote explaining the abbreviations used in a crosstable

Use it below
[`body_add_crosstable()`](https://danchaltiel.github.io/crosstable/reference/body_add_crosstable.md).
Footnote: Med: median, IQR: interquartile range, Std: standard
deviation. Percentages are expressed in column.

## Usage

``` r
body_add_crosstable_footnote(doc)
```

## Arguments

- doc:

  a `rdocx` object

## Value

The docx object `doc`
