# Default function to display the effect

User can provide their own custom version in
[`crosstable_effect_args()`](https://danchaltiel.github.io/crosstable/reference/crosstable_effect_args.md)

## Usage

``` r
display_effect(effect, digits = 4)
```

## Arguments

- effect:

  effect

- digits:

  digits

## Value

a character vector
