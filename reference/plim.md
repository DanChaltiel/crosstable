# Format p values (alternative to [`format.pval()`](https://rdrr.io/r/base/format.pval.html))

Format p values (alternative to
[`format.pval()`](https://rdrr.io/r/base/format.pval.html))

## Usage

``` r
plim(p, digits = 4)
```

## Arguments

- p:

  p values

- digits:

  number of digits

## Value

formatted p values
