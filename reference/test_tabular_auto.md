# test for contingency table

Compute a chisq.test, a chisq.test with correction of continuity or a
fisher test as appropriate

## Usage

``` r
test_tabular_auto(x, y)
```

## Arguments

- x:

  vector

- y:

  another vector

## Value

a list with two components: p.value and method
