# Replace text on several bookmarks at once

Replace text on several bookmarks at once

## Usage

``` r
body_replace_text_at_bkms(doc, ..., envir = parent.frame())
```

## Arguments

- doc:

  a `rdocx` object

- ...:

  named

- envir:

  Environment to evaluate each expression in `glue()`.

## Value

The docx object `doc`
