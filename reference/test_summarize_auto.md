# test for mean comparison

Compute a oneway.test (with equal or unequal variance) or a kruskal.test
as appropriate.

## Usage

``` r
test_summarize_auto(x, g)
```

## Arguments

- x:

  vector

- g:

  another vector

## Value

a list with two components: p.value and method
